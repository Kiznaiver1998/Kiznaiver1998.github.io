我们常常会使用Array.map()，Array.filter()和Array.findIndex()来对对象数组进行分组，但其实呀，JavaScript的Array对象中还有很几种其他的方法可以达到效果。

在深入研究使用Array.reduce()创建分组函数groupBy前，我们将看到几个基本的Array.reduce示例

reduce函数是怎样使用的

简单的来说，它将数组分成单个的值，这个单个的值可以是Number, String, Object 或者是其他数组

（Ps：译者「我」的话: https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/Reduce 不是很明白的小伙伴也可以看看MDN）

让我们康康一个常见reduce的使用

  • reduce接受一个初始值，在我们的年龄加法中，我们设置了 0 作为初始值（译者「我」的话: 可以看到reduce的第2个参数就是初始值）
  • reduce会遍历数组的每个值，并且每次把这个值和之前的得到总和相加。因此，我们每一步会返回综合，并且这个总和会与下一个年龄值相加
  • 在迭代（遍历）完成后，最终的总和值也将会返回。
