三角形实现

我们用了 4 个实例来演示一下三角形实现的原理，其实是利用 border 实现的一个 CSS Trick

当 content 有宽高时，border 就为四个等腰梯形拼接，如果让 3 条边框颜色设置为 transparent 那么就能使用这个技巧获得一个等腰梯形；而当 content 没有宽高时，border 就为四个等腰三角形拼接。如果让 3 条边框颜色设置为 transparent 那么就能使用这个技巧获得一个等腰三角形；

需要注意的是：自己单写的时候需要将三角形 width: 0;
