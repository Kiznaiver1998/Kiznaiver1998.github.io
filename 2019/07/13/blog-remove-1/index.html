遇到的问题：

https://cloud.tencent.com/document/product/213/10912

搭建Ftp服务器后,输入telnet + 云服务器公网 IP + 21也能够成功连接。

但是在FileZilla连接时出现错误如下

解决方案

我随后百度有以下几个解决方案：

https://blog.csdn.net/shuaicenglou3032/article/details/72603131
